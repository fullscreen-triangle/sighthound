  Accuracy: 50-100 meters (best with reanalysis data ERA5, MERRA-2)

  Disadvantages: Coarse resolution, requires model calibration

REGIME 2: AIRPORT/URBAN INFRASTRUCTURE
  Purpose: Dense local positioning in cities and airports
  Method: Measure transfer-matrix rank from multi-frequency signals
  Theory: rank(M(p)) = min(N_sensors, K_frequencies) at true position
  Accuracy: 1-10 meters in dense areas
  Cost: Free (signals already transmitted)
  Equipment: None (tap existing antenna feeds)
  Implementation: Airports have 10-20 antennas × 10+ frequency bands
  Advantages: High accuracy, passive observation
  Disadvantages: Requires infrastructure; sparse in rural areas

REGIME 3: CELESTIAL POSITIONING
  Purpose: Unjammable global positioning using natural starlight
  Method: Measure harmonic coupling from 4+ celestial sources
  Theory: Position uniquely determines observed harmonic signatures
  Accuracy: 10-100 meters globally
  Cost: ~$30 for optical filter (one-time)
  Equipment: Smartphone camera + daytime filter + free software
  Implementation: Identify stars with Stellarium/SkySafari, solve triangulation
  Advantages: Unjammable, unspoof-able, covert, global, requires no infrastructure
  Disadvantages: Requires clear sky (~40-60% availability), 5-10 min measurement

THREE PRINCIPAL THEOREMS
================================================================================

THEOREM 1: HARMONIC SIGNATURE UNIQUENESS
  Statement: For distinct positions p₁ ≠ p₂, observed harmonic eigenvalues σ(p₁) ≠ σ(p₂)
  Proof: Uses smoothness of environmental fields (B_g, g, u, ρ) and continuity of eigenvalues
  Implication: Position uniquely determines harmonic fingerprint (topological invariant)
  Application: Measuring eigenvalues → determines position

THEOREM 2: INFRASTRUCTURE RANK CONSTRAINT
  Statement: rank(M(p)) = min(N, K) at any location (under generic conditions)
  Proof: Transfer operator limited by min(N sensors, K frequencies)
  Implication: Rank provides independent position constraint
  Application: Searching position space for location maximizing rank match

THEOREM 3: CELESTIAL TRIANGULATION
  Statement: Four or more celestial sources with measured harmonics uniquely determine 3D position
  Proof: Overdetermined system (4 equations, 3 unknowns) has unique least-squares solution
  Implication: Stars provide unjammable position determination
  Application: Smartphone celestial app automatically determines global position
